# Secure Drop Messenger

Secure Drop — это безопасное приложение для обмена сообщениями с end-to-end шифрованием, разработанное для обеспечения приватности и анонимности. Оно имеет современный интерфейс в стиле glassmorphism и работает как прогрессивное веб-приложение (PWA).

## Ключевые особенности

* **Сквозное шифрование (E2EE)**:
    * Сообщения и медиа шифруются ключами AES-GCM (256-bit) на устройстве отправителя.
    * Приватные ключи (ECDH P-256) хранятся в зашифрованном виде (AES-GCM) на сервере и расшифровываются только паролем пользователя.
    * Защита от подмены ключей при входе: автоматическая проверка принадлежности ключей устройству.
* **Анонимность**: Не требуется номер телефона или email. Регистрация по Username + Password (для восстановления) + TOTP (для 2FA).
* **Безопасное восстановление**: Если вы потеряли устройство, чаты можно расшифровать, введя Пароль Восстановления (который расшифровывает ваш приватный ключ с сервера).
* **Защищенный обмен**: Полный запрет на отправку незашифрованных (plaintext) сообщений на уровне кода.
* **Smart Key Management**: Автоматическое обнаружение подмены ключей и защита от сбоев (Key Sanitization).
* **В реальном времени**: Мгновенная доставка и индикация статуса.
* **PWA & Mobile Ready**: Работает как приложение на iOS/Android (устанавливается через "Add to Home Screen").
* **Современный UI**: Glassmorphism дизайн, адаптивность, плавные анимации.

## Технологический стек

* **Frontend**: React, TypeScript, Vite, Tailwind CSS
* **Backend**: Python, FastAPI, SQLAlchemy
* **База данных**: PostgreSQL
* **Real-time/Кэш**: Redis
* **Контейнеризация**: Docker & Docker Compose

## Быстрый старт (локальная разработка)

1. Клонируйте репозиторий:

```bash
git clone https://github.com/yourusername/secure_drop.git
cd secure_drop
```

2. Запустите с помощью Docker Compose:

```bash
docker compose up --build
```

3. Откройте приложение:
   * Frontend: http://localhost:3000 (или порт 80 в зависимости от конфигурации)
   * Backend API: http://localhost:8000/docs

## Структура проекта

* `/backend`: FastAPI приложение, модели и API-роутеры.
* `/front`: React frontend приложение.
* `/docker-compose.yml`: Оркестрация для БД, Redis, Backend и Frontend.
